
### 源码学习

1. 安装依赖
    ```
    yarn install
    ```

2. 启动 dev server
    ```
    yarn server
    ```
### 项目介绍
该项目是参照Ant Design、ElementUI等UI库，使用Vue.js实现的简易UI组件库，目前已支持按钮、输入框、网格、布局、弹窗、页签、冒泡、折页等组件。每个组件均经历从需求分析到 TravisCI实现持续集成，最后以VuePress为基础制作官网的工程化流程。
### 技术栈
Vue.js/ VuePress/ ES6/ Parcel/ Npm Scripts/ SCSS/ TravisCI
